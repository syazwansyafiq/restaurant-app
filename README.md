## About This Repo

This is an example of a restaurant food ordering system API. 

## To Get Started

<ul>
    <li> Composer install </li>
    <li> npm install </li>
    <li> php artisan migrate </li>
    <li> php artisan db:seed </li>
    <li> php artisan serve </li>
    <li> npm run dev </li>
    <li> php artisan queue:work </li>

</ul>
